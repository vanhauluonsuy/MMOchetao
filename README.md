## 📌 English Description

**MMOCheTao** is a Minecraft plugin that allows players to craft **random MMOItems** through a custom graphical interface (GUI).

Players exchange required materials to receive **random items defined in MMOItems**, creating a fun, unpredictable, and balanced crafting experience.

This plugin is designed for RPG, Survival, and MMO-style servers that want to expand crafting mechanics using MMOItems.

---

## 📌 Mô tả Tiếng Việt

**MMOCheTao** là plugin Minecraft cho phép người chơi **chế tạo item ngẫu nhiên từ MMOItems** thông qua **GUI trực quan**.

Người chơi chỉ cần đủ nguyên liệu, plugin sẽ **random item** theo cấu hình đã thiết lập sẵn, giúp gameplay trở nên thú vị và không nhàm chán.
